# 🚀 Quick Start - Optimized for Fast Demos

Run TradingAgents in **1.5-2 minutes** with all 4 analysts!

---

## ⚡ Fastest Way to Start

```bash
./start.sh
```

Then open: **http://localhost:3000**

That's it! ✅

---

## 🎯 What You'll See

1. **Enter stock ticker** (e.g., NVDA)
2. **All 4 analysts are pre-selected:**
   - 📈 Market Analyst
   - 💬 Social Analyst  
   - 📰 News Analyst
   - 📊 Fundamentals Analyst
3. **Click "Start Analysis"**
4. **Watch real-time updates** (~1.5-2 minutes)
5. **Get BUY/SELL/HOLD decision** with complete reasoning
